# Introduction to Deno

Deno is a simple, modern, and secure runtime for JavaScript and TypeScript that uses V8 and is built in Rust.

## Key Features

- Secure by default
- Built-in TypeScript support
- Single executable
- Standard library
- Modern JavaScript features
